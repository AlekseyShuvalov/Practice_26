# Practice_26
Репозиторий для выполнения учебной практики.

## Постановка задачи
Дана полиномиальная функция $f(x)$ степени <= 8. Необходимо найти параметры ступенчатой функции $g(x)$ с заданным количеством ступеней k на интервале [l, r] для приближения исходной полиномиальной функции.

## Представление данных
1. **Целевая функция $f(x)$**: используется массив вещественных чисел для хранения коэффициентов функции, где индекс элемента соответствует степени x.
2. **Ген**: вещественное число, равное высоте одной ступени.
3. **Генотип**: одномерный массив вещественных чисел, длина которого равна количеству ступеней. Каждый генотип является полноценной ступенчатой функцией.
4. **Популяция**: двумерный массив, состоящий из N генотипов.

## Функция качества
1. **Расчёт ошибки**: $Error = \frac{1}{N} \sum_{i=0}^{N} |f(x_i) - g(x_i)|$
2. **Приспособленность**: $Fitness = \frac{1}{1+Error}$

Для имитации эволюции будут использоваться: турнирный отбор для трёх генотипов, одноточечное скрещивание и мутация с заданной вероятностью.

## Общий алгоритм
1. **Ввод данных**: пользователь программы вводит коэффициенты полиномиальной функции, интервал, количество ступеней и параметры генетического алгоритма (размер популяции, число поколений и шанс мутации). Доступен ручной ввод, случайная генерация данных, чтение из файла.
2. **Генерация начальной популяции**
3. **Цикл эволюции**
	* Оценка приспособленности для каждой особи в популяции.
	* Сохранение лучшей особи.
	* До тех пор, пока не заполнится новая популяция:
		* Турнир для выявления родителей.
		* Скрещивание.
		* Мутация потомков.
		* Добавление потомков в новую популяцию.
	* Замена старой популяции на новую.
4. **Вывод результата**: лучший найденный генотип содержит искомые высоты ступеней.

## Приветственное окно
Будет отображаться при запуске программы. Есть возможность выбрать 4 опции: 
* Задать данные через графический интерфейс
* Загрузить файл с заранее прописанными параметрами
* Сгенерировать случайные параметры
* Просмотр информации о программе

## 1. Окно настройки задачи
Ввод параметров для задачи. 
Возможность вернуться на прошлое окно или перейти на следующее.


## 2. Окно с загрузкой файла
Выбрать файл, который хранит в себе все параметры для задачи.  Возможность вернуться на прошлое окно или перейти на следующее.


## 3. Окно с информацией
Указание авторства; постановка задачи; язык программирования.
Возврат на главное окно


## 4. Окно выполнения алгоритма
В левой части указана информация о текущем этапе: 
* Текущее поколение 
* Лучшая ошибка текущего поколения

В правой части изображены два графика:  
* График исходного полинома и текущей ступенчатой функции, который обновляется после каждого поколения
* График изменения функции качества в зависимости от номера поколения. График дополняется на каждом шаге выполнения алгоритма

В нижней части 3 кнопки:   
* Назад: для возврата на предыдущее поколение
* Вперед: для перехода на следующее поколение
* В конец: для перехода на самое последнее поколение
